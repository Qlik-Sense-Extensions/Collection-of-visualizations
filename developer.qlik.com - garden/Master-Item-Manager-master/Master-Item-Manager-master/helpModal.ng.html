Extension Instructions:


  1. Import your metrics using the provided excel file.
  2. In your Sense application create 2 Tables: One that contains the Measures and one that contains Dimensions from the imported excel file.
  3. Selecting a subset of Dimensions or Measures from the Table will allow the Manager app to only perform actions on those selected (No selections means everything is selected)
  4. Actions will only be performed on the Master Items defined in the template sheet (Self defined metrics will be unaffected).
