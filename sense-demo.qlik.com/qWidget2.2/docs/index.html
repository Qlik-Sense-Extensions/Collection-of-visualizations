Introduction

The basic idea behind qWidgets is that you are able to create new visualizations without the need of programming in JavaScript.

By just using HTML & CSS you can create completely new and astonishing objects in Qlik Sense which behave like all other objects and even more:

qWidgets can be

  • Created by just copying HTML & CSS from one of the thousands of libraries out there ...
  • Copied from one app to another app
  • Put and to master-library and re-used from there
  • Used in Story Telling
  • Used across apps and even environments of Qlik Sense
